# 💎 こえの宝石工房(こえの結晶ラボ)

**声を出すと、洞窟の中で宝石が少しずつ育つ 3D ゲーム**です。
4歳のお子さまが iPhone / iPad(縦画面・横画面どちらでも)で遊ぶことを想定しています。

「あーー」と言えば丸くてももいろの結晶が、「いーー」と言えば細くてそらいろの結晶が生えてきます。
**同じ声の組み合わせからは、いつも同じ宝石が育ちます**(決定論的生成)。
7回声を届けると宝石が完成し、名前がついて「ほうせきのたな」にコレクションされます。

---

## あそびかた

1. `index.html` を開いて「**はじめる ✨**」をタップ
2. マイクの使用を許可する(許可しなくても 🎹 の「こえボタン」で遊べます)
3. 台座の上の小さな結晶に向かって、声を出す
   - 声を出している間は、ようせいの**ルミ**が口を開けて聞いてくれて、
     これから生える結晶のプレビューがキラキラと現れます
4. 声をやめると結晶がポンッと固まり、世界が変わります
   - 壁の結晶が同じ色で点灯する
   - 足もとの水に宝石の色が溶けて波紋が広がる
   - まわりに花や小さな結晶が咲く
   - 宝石から光の粒が立ちのぼる
5. 上の 7 つの丸(できたよメーター)が全部うまると **かんせい!**
   花火とファンファーレでお祝いして、たなに保存されます

## 音と宝石の対応表

| 音 | 宝石の変化 |
| --- | --- |
| あーー | 丸くふくらむ。ピンク・赤・オレンジ系 |
| いーー | 細く鋭い結晶が伸びる。青・水色・銀色系 |
| うーー | ぽってりした雫型。紫・藍色 |
| えーー | 枝分かれした結晶。緑・黄緑系 |
| おーー | 大きな輪と金色の宝玉 |
| 高い声 | 上のほうに付き、透明度が上がる |
| 低い声 | 下のほうに付き、色が濃くなり、根が伸びる |
| 短い声 | 小さな星の粒が入る |
| 長い声 | 大きく育ち、外側にもう一枚の層ができる |
| 手拍子 2回 | 2本の光る筋が入る |
| 手拍子 3回 | 三角形・三つ葉の模様になる |
| ふー(息) | 雪の結晶と白いくもりガラスになる |

## 起動方法

マイク(`getUserMedia`)を使うため、**HTTPS または localhost** で開く必要があります。

### ローカルで試す

```bash
cd koe-no-hoseki-koubou
python3 -m http.server 8000
# → ブラウザで http://localhost:8000 を開く
```

### iPhone / iPad で遊ぶ

GitHub Pages などの HTTPS ホスティングにこのフォルダを置いて、Safari で開いてください。
ホーム画面に追加するとフルスクリーンで遊べます。

- 効果音を鳴らすため、消音スイッチ(サイレントモード)は解除してください
- 静かな部屋のほうが認識が安定します

## マイクが使えないとき

左下の 🎹 ボタンで「こえボタン」が開きます。
「あ・い・う・え・お」を**ながおし**すると長い声に、上の「たかい・ふつう・ひくい」で声の高さを選べます。
「ふー」「👏👏」「👏👏👏」ボタンもあります。マイク許可が取れなかった場合は自動で開きます。

## おうちのかたへ(プライバシー)

- マイクの音は**その場で解析するだけ**で、録音・保存・送信は一切しません
- 作品(レシピとサムネイル)はこの端末の `localStorage` にだけ保存されます
- 画面右上の 🛈 から、おしゃべり(音声合成)の ON/OFF と作品の全削除ができます

## 技術構成

| 項目 | 内容 |
| --- | --- |
| 3D 描画 | [Three.js](https://threejs.org/) r160(`js/lib/` に同梱・外部 CDN 不要) |
| 音声解析 | Web Audio API(ライブラリ不使用) |
| ピッチ検出 | 正規化自己相関(NSDF、McLeod 風) |
| 母音判定 | FFT スペクトル包絡からフォルマント F1/F2 を推定し、5母音の基準値と対数最近傍で分類 |
| 手拍子検出 | 音量スパイク+低周期性のトランジェント検出、時間窓で回数を確定 |
| 息(ふー)検出 | 非周期ノイズが一定時間続くことで判定 |
| 決定論的生成 | 入力列(レシピ)の内容から seed を作る mulberry32。同じレシピ → 同じ宝石 |
| 効果音 | Web Audio によるリアルタイム合成(音声ファイルなし)+ 音声合成のほめことば |
| 保存 | `localStorage`(レシピ+JPEG サムネイル、最大24個) |

## ファイル構成

```
koe-no-hoseki-koubou/
├── index.html            エントリポイント
├── README.md             このファイル
├── css/style.css         4歳児向け UI(縦横対応・safe-area 対応)
└── js/
    ├── main.js           画面遷移・成長ロジック・メインループ
    ├── config.js         音→宝石の対応表・しきい値(調整はここ)
    ├── audio/
    │   ├── mic.js        マイク取得と毎フレームの特徴量
    │   ├── pitch.js      ピッチ検出・ゼロ交差率
    │   ├── vowel.js      フォルマント推定と母音分類
    │   ├── events.js     声・手拍子・息のイベント状態機械
    │   └── sfx.js        効果音合成・ファンファーレ・おしゃべり
    ├── game/
    │   ├── rng.js        決定論的乱数(mulberry32)
    │   ├── recipe.js     レシピ(入力列)の定義とシリアライズ
    │   ├── gemBuilder.js レシピ → 宝石3Dモデル(形の実装はここ)
    │   ├── world.js      洞窟・水・壁の結晶・花・ほたる
    │   ├── particles.js  キラキラ・光の流れ・花火・霧
    │   ├── character.js  ようせいルミ(声に反応する顔つき)
    │   └── gallery.js    保存・なまえ付け・サムネイル描画
    ├── ui/hud.js         DOM の HUD(たな・お祝い・こえボタン)
    └── lib/              Three.js 本体(MIT License)
```

## 開発者向け: テストモード

`index.html?test=1` で開くと `window.__game` にフックが生えます。
`window.__game.demo()` を実行すると「あ→い→う→え→お→ふー→👏👏👏」の
自動デモが走り、完成まで確認できます(マイク不要)。
