# こえの魔法筆（まほうふで） 🖌️✨

**モノクロの3Dせかいを、こえで染める** — 4歳児向けの音声お絵かきゲーム。

指でなぞりながら「あーー」「いーー」と声を出すと、その声が“絵の具”になって、
灰色の世界に色がじゅわっと染みていきます。声を出している間じゅう、世界は変わり続けます。

iPhone / iPad の Safari（縦画面・横画面どちらでも）で遊べます。

---

## あそびかた

1. **はじめる** をタップ（マイクの許可を聞かれたら OK）
2. すきな世界をえらぶ（にわ・すいそう・もり・おしろ・まち・へや の6つ）
3. **指でなぞりながら、こえを出す**。それだけ！
   - 指をはなしていても、ようせいの筆が代わりに塗ってくれます
   - 2本指で、きょうだい一緒にも塗れます
4. 世界じゅうが色づくと、花火のおいわい 🎆

## こえと魔法の対応

| こえ | まほう |
| --- | --- |
| あーー | 赤・ピンクの丸いにじみが じゅわっと広がる |
| いーー | 青・水色の光の線と星が しゅんと走る |
| うーー | 紫・深い青の泡が ぷくぷく出る（はじけると音がなる） |
| えーー | 緑の葉っぱとツタが ぐんぐん伸びる |
| おーー | 金色の大きな輪が わーんと広がる |
| ふーー（息） | キラキラの風が広いはんいを ふわっと染める |
| ぱっ！（短く） | スタンプ！ 近くのものが一気に色づく |

さらに——

- **こえの長さ**: 長く出すほど、染みが大きく育つ
- **こえの高さ**: 高い声はうえへ・明るく、低い声はしたへ・深く
- **高さの動き**: 「あ↗︎」でツタや星がのぼり、「あ↘︎」でおりる
- **リズム**: 「あ、あ、あ」と区切ると、回数ぶん花びらが ぱっぱっぱっ
- **こえの揺れ**: 「あ〜あ〜あ〜」でマーブルの波もよう
- 色がついたものは **目をさまして動き出す**（魚が泳ぐ、ちょうが舞う、車が走る…）

## 音量はノーカウント 🤫

このゲームは **声の大きさを一切スコアに使いません**。

- 音量は「声が出ているかどうか」の検出だけに使用（ノイズフロアを自動学習）
- ささやき声でも、大声でも、出る魔法はまったく同じ
- 魔法の変化を決めるのは 母音・高さ・長さ・リズム・声質 だけ

「大きな声を出して！」ではなく **「ちいさな声でも魔法が出るよ」** の設計です。

## マイクがなくても遊べます

マイクが使えない場合は、画面の「あ・い・う・え・お」ボタンを**押しながら**なぞれば同じ魔法が出ます。
ボタンの上のほうを押すと高い声、下のほうを押すと低い声の扱いになります。
（デスクトップでは A / I / U / E / O キーでも代用できます）

## 動かしかた

ビルド不要の静的サイトです（Three.js は `vendor/` に同梱、外部CDNなし・完全オフライン動作）。

```bash
cd koe-no-mahou-fude
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

> **注意**: マイク（getUserMedia）はセキュアコンテキストが必要です。
> `localhost` 以外から使う場合は **HTTPS** で配信してください（GitHub Pages が手軽です）。
> リポジトリの Pages を有効にして `koe-no-mahou-fude/` を公開すれば、iPhone/iPad からそのまま遊べます。

## プライバシー

- 声はすべて端末内の Web Audio API で解析され、**録音・保存・送信は一切ありません**
- 通信も発生しません（進行状況は localStorage にのみ保存）

## 技術メモ

```
koe-no-mahou-fude/
├── index.html            画面構成・UI
├── css/style.css         幼児向け大ボタンUI（縦横両対応、セーフエリア対応）
├── assets/icon.svg       アイコン
├── vendor/three.module.js  Three.js r160（同梱）
└── js/
    ├── main.js           画面フロー・保存
    ├── voice/
    │   ├── analyzer.js   声の解析エンジン
    │   └── soundfx.js    WebAudioで合成する効果音（音声ファイル不使用）
    └── game/
        ├── game.js       ゲーム本体（入力・塗り・アニメ・おいわい）
        ├── reveal.js     モノクロ→カラーの「染み」シェーダー
        ├── effects.js    母音ごとのパーティクル魔法
        ├── worlds.js     6つの手作り3Dワールド
        └── ui.js         HUD・ようせいメッセージ・母音ボタン
```

- **母音判定**: FFTスペクトルから第1・第2フォルマント（F1/F2）を推定し、
  幼児声域向けにチューニングした母音セントロイドとの対数周波数距離で分類。時間方向に多数決で平滑化。
- **ピッチ**: 自己相関法（100〜700Hz）。上昇/下降トレンドは直近0.5秒の回帰傾き。
- **息の検出**: スペクトル平坦度 × 周期性の低さ。**破裂音**: 鋭いアタック × 0.2秒以内の短発声。
- **声の揺れ**: ピッチ履歴の3〜8Hz振動を検出。
- **染みシェーダー**: 各メッシュのマテリアルに `onBeforeCompile` でパッチを当て、
  「塗りスポット（中心＋成長する半径）」の内側だけ本来の色を表示。境界は現在の母音の色でゆらゆら光る。
- **効果音**: すべてWebAudioでリアルタイム合成。子どもの声の高さをペンタトニックに量子化して鳴らすので、
  どんな声でもゲームと“合奏”になる。
