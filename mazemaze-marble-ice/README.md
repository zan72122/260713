# まぜまぜマーブル・アイス 🍨

4歳児向けの、アイスをぐるぐる混ぜて遊ぶゲームです。
iPhone / iPad の縦画面・横画面のどちらでも遊べます。文字は使わず、絵アイコンだけで操作できます。

## あそびかた

1. はじまると、毎回ちがうフレーバーのアイスが2〜3個ランダムな場所に「ぽとん」と落ちてくる(リセット🔄でまっさらのお皿に戻せる)
2. 上のバーからすきな色のアイスを足して、指でお皿の上をぐるぐる混ぜる → マーブル模様ができる
3. フレーバーごとに「さわりごこち」がちがう。混ぜると質感どうしも本当に混ざり合う:

| フレーバー | 固有の質感 |
| --- | --- |
| 🍓 いちご(赤) | **ぷるぷる** — さわると波打つゼリー。つやつや透きとおる |
| 🥤 ソーダ(青) | **しゃりしゃり** — シャーベットの粗い氷粒。青白くきらめき、混ぜるとしゃりしゃり鳴る |
| 🍦 バニラ(白) | **ふわふわ** — 空気たっぷり。マットで白く、ふんわり盛り上がる |
| 🥭 マンゴー(黄) | **もちもち** — のびる粘弾性。筋が長くくっきり伸び、にじみにくい |
| 🍫 チョコ(茶) | **ねっとり** — つやが持続する濃厚な質感。混ぜるとねちゃっと鳴る |
| 🌿 チョコミント(緑) | **ぱりぱり** — チョコ殻コーティング。押して混ぜるとぱきぱき割れて中の色がのぞく。冷やすと固まり直す |

4. 混ぜ方・温度でもようすが変わる:

| 混ぜ方 | 結果 |
| --- | --- |
| ゆっくり1回だけ混ぜる | 太いマーブル模様 |
| 細かく何度も混ぜる | 全体がひとつの色に近づく |
| スプーン/ヘラで押しながら混ぜる | 空気が抜けて表面につやが出る |
| ❄️で冷やしながら混ぜる | 高く盛り上がってゴツゴツ角が立ち、氷の粒がきらきら光る |
| ☀️であたためながら混ぜる | 平たくとろけて縁がだらっと広がり、液体っぽく流れる |
| 🍪クッキーを入れて混ぜる | 粒がわれて残り、断面が汚くかわいい |

### ボタン

- **上バー**: フレーバー6種(いちご・ソーダ・バニラ・マンゴー・チョコ・チョコミント) / トッピング3種(クッキー・チョコチップ・いちご果肉)
- **下バー**: 道具(指・スプーン・ヘラ) / 温度(❄️冷やす・☀️あたためる ※長押し) / 音のオンオフ / リセット(タップでお皿がまっさらに)

## 技術

外部ライブラリ・外部音源なしの完全スタンドアロン(静的ファイルのみ)。

### アイスの質感 — WebGL2 流体シミュレーション

各ピクセルが色に加えて次の状態を持ち、GPU上で移流・発展します。

| 状態テクスチャ | 内容 |
| --- | --- |
| velocity (RG16F) | 流れの速度場。温度・質感依存の粘性減衰 + 圧力投影(非圧縮) |
| color (RGBA16F) | フレーバー色 + アイスの量(高さの元) |
| props (RGBA16F) | 温度 / 空気量 / 氷結晶サイズ / 表面ツヤ |
| props2 (RGBA16F) | しゃりしゃり / もちもち / ぱりぱり殻 / ぷるぷる(フレーバー固有の質感) |

- **マーブル模様**: MacCormack法の移流で筋をくっきり保持。細かく混ぜ続けると移流に拡散が混ざり均一色へ
- **温度と形**: キンキンだと高く盛り上がって輪郭が立ち、ゴツゴツした起伏が出る。とろけると平たく広がり縁がだらける
- **ふわふわ**: 空気量が高さと拡散光を持ち上げ、マットで白っぽい質感に
- **ざくざく**: 氷結晶量がスペックルノイズ・法線の凹凸・またたくグリントになる
- **しゃりしゃり**: 粗い氷粒の起伏 + 青白いきらめき。冷やすと粒が育ち直し、熱でとける
- **もちもち**: 粘性減衰を下げて流れが長く伸び、MacCormackのにじみを抑えて筋がくっきり残る
- **ぱりぱり**: チョコ殻。押し混ぜで割れて(ヒビ模様から中の色がのぞく)、冷やすと固まり直す
- **ぷるぷる**: 速度場に反応して表面が波打つ。透きとおった色とつや
- **ねっとり/つや**: 押し混ぜで空気が抜けてツヤ(スペキュラ)が上がる。もちもち領域はツヤが持続
- **とろとろ**: 温度が上がると粘性が下がり、高さ勾配を下る「たれ流れ」でシロップ状に広がる

フレーバー固有の質感は色と同じ場を移流するため、混ぜるとマーブル模様と一緒に質感も本当に混ざり合います。

### サウンド — Web Audio APIによる全合成

- 混ぜ音はフレーバー連動: 指の下の質感をGPUから読み出し(2×1ピクセルのreadPixels)、
  ざくざく / しゃりしゃり / ねっとり / もちもち / ぷるぷる / とろとろをリアルタイムにブレンド
- チョコ殻を割ると「ぱりぱりっ」と連続クラック音
- オルゴール調BGM + やわらかいパッドをスケジューラで生成
- ぽとん / パラパラ / ぱきっ / ひんやり / ぽかぽか などのワンショットSFX

### トッピング粒

CPUパーティクル(最大260個)をGLポイントスプライトで描画。かき混ぜの力で流され、
混ぜるほど沈んでまわりのアイス色をまとう。クッキーは力がかかると割れて小さくなる。

## ファイル構成

```
mazemaze-marble-ice/
├── index.html
├── README.md
├── assets/icon.svg
├── css/style.css
└── js/
    ├── main.js         … 起動・入力・ゲームループ・ランダム初期配置
    ├── sim.js          … 流体シミュレーション(パス構成・質感サンプリング)
    ├── shaders/
    │   ├── common.js   … 共通GLSL部品(ノイズ・お皿マスク等)
    │   ├── sim.js      … スプラット・移流・圧力・質感の時間発展
    │   ├── render.js   … 最終レンダリング(質感の見た目)
    │   └── particles.js… トッピング粒・雪・湯気のシェーダー
    ├── particles.js    … トッピング粒・雪・湯気(CPU側)
    ├── audio.js        … 効果音の合成(質感連動ミキサー)
    ├── audio_bgm.js    … オルゴール調BGM
    ├── ui.js           … アイコンボタンUI・フレーバー定義
    └── gl.js           … WebGL2ヘルパー
```

## 動作環境

WebGL2 と `EXT_color_buffer_float` が必要です(iOS 15以降のSafari、モダンなChrome/Edge/Firefoxで動作)。
ローカルで遊ぶには静的サーバーで配信してください:

```
python3 -m http.server 8000
# → http://localhost:8000/mazemaze-marble-ice/
```
